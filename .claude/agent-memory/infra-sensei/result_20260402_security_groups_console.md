📚 今日の学習ポイント
- セキュリティグループはステートフル：インバウンドで許可した通信の応答は自動で返る
- SGを別のSGのソースに指定する「SG参照」により、動的IPでも安全なアクセス制御ができる
- 最小権限の原則：ALBはインターネットから、ECSはALBから、DBはECSからのみという連鎖構造

⚠️ 要注意ポイント（苦手分野）
- VPC選択の確認（SGはVPCに紐づき、後から変更不可）
- インバウンドルールのソースをIPではなくSG IDで正確に指定すること
