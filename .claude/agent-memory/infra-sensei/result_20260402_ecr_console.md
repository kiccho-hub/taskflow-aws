## 今日の学習ポイント

- ECR はプライベートな「Dockerイメージの保管庫」で、ECS Fargate がここからイメージを pull して起動する流れを理解できた
- ライフサイクルポリシー = 「古いイメージを自動で捨てる掃除ロボット」というコスト管理の考え方を実践できた
- タグのイミュータビリティ = 「一度貼ったラベルは剥がせない」ことで本番イメージの追跡可能性を保証できることを学んだ

## 要注意ポイント（苦手分野）

- 複数リポジトリへの「同じ設定の繰り返し適用」で漏れが起きやすい
- URI のメモ・確認など「後で使うもの」の事後確認習慣がまだ定着途上
