## 今日の学習ポイント

- VPCとは「AWSクラウド上の自分専用の仮想ネットワーク（会社の専用ビル）」であること
- IGWは「門」、ルートテーブルは「道路地図」という役割の違いを体験できました
- プライベートサブネットにパブリックIPを付けない理由（セキュリティの基本）を実践で学びました

## 要注意ポイント（苦手分野）

- NATゲートウェイのコスト管理（削除忘れに注意）
- ルートテーブルとサブネットの関連付けを忘れると通信できない点
