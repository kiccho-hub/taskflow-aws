🔧 改善ポイント（Try）
- IAMロール作成時に「Elastic Container Service Task」を選ぶことが重要なポイントです。「ECS」そのものではなく「Task」を選ばないとコンテナに権限が付与されません
- Container Insights は有効化したかどうかを確認する習慣をつけましょう。コスト増加を気にして無効のまま進んでしまうケースがよくあります

📚 今日の学習ポイント
- ECS の階層構造（Cluster → Service → Task）を体で覚えられました
- Fargateを選ぶことでサーバー管理が不要になる、という「サーバーレス」の考え方を実感できたと思います
- IAMの「最小権限の原則」を ECSTaskExecutionRole を通じて学べました

⚠️ 要注意ポイント（苦手分野）
