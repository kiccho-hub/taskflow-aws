📚 今日の学習ポイント
- CloudWatchダッシュボードは「車のメーターパネル」— 複数の計器を一画面で監視できる
- アラームは「温度計のアラーム機能」— 閾値を超えたら自動で知らせてくれる
- SNS＋CloudWatchの組み合わせで「問題発生→即座に通知」のサイクルを自動化できる

⚠️ 要注意ポイント（苦手分野）
- Insufficient dataは「まだデータが溜まっていない」状態であり、エラーではない点
- アラームのしきい値設定は実際のワークロード特性に合わせる必要がある
